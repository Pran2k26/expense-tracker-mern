# Expense Tracker MERN App

A simple **Finance Tool** built with the *MERN stack* (**MongoDB, Express.js, React.js, Node.js**) to help users track their expenses, view, filter, and sort them by category and date.  

This project demonstrates **full-stack development**, REST API integration, and frontend-backend interaction in a real-world-like application.
